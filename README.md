# 🔥 NoteForge

**NoteForge** is a full-stack MERN-based note management platform that allows users to create, read, update, and delete notes seamlessly. Built with scalability in mind, it is designed to evolve into an AI-powered productivity tool.

---

## 🚀 Features

- ✍️ Create, Read, Update, Delete (CRUD) notes  
- ⚡ Fast and responsive UI  
- 🔗 RESTful API integration  
- 🧩 Modular and scalable backend  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
